Chapter IV: NLP Text Classification

July 02, 2022

  • Created a model that classifies the Spanish news articles.

  • To solve this problem, I have used two different Algorithms/Frameworks. 1. Used Sagemaker Built-in Algorithm Blazing text is used for Text classification. 2. we have used Sagemaker SKlearn Framework. The SKlearn framework is used so that we can use machine learning algorithms in SageMaker. like we have used LogisticRegression.

  • Data Preprocessing Steps: Porter Stemming, Converting TO LowerCase, Removing Punctuation and Low frequency term filtering (count < 3).

  • Store Data in s3 Bucket

  • Hosting Data In Prodection Environment in Sagemaker instance_type='ml.t2.medium'
